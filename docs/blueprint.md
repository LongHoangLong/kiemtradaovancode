# **App Name**: CodePlagiarismCheck

## Core Features:

- Assignment Upload: Cho phép giảng viên tải lên một tệp nén (.zip hoặc .rar) chứa các bài nộp code của sinh viên.
- Language Detection: Tự động phát hiện ngôn ngữ lập trình (C# hoặc Python) của mỗi tệp đã nộp.
- Token-Based Plagiarism Detection: Triển khai thuật toán phát hiện đạo văn dựa trên token để so sánh các bài nộp code.
- Plagiarism Report Generation: Tạo báo cáo đạo văn chi tiết, làm nổi bật các trường hợp đạo văn tiềm ẩn và tỷ lệ phần trăm tương đồng code giữa các bài nộp.
- Memvid Integration: Tích hợp với Memvid (hoặc một giải pháp lưu trữ mã nguồn mở khác) để lưu trữ các bài tập đã tải lên và dữ liệu phát hiện đạo văn.
- User Interface Language Selection: Người dùng chọn ngôn ngữ giao diện mà họ muốn sử dụng: tiếng Anh hoặc tiếng Việt.

## Style Guidelines:

- Màu chính: Màu xanh lam chuyên nghiệp (#29ABE2) để truyền tải sự tin cậy và chính xác.
- Màu nền: Màu xám nhạt (#F2F2F2) cho giao diện sạch sẽ, không lộn xộn.
- Màu nhấn: Màu cam (#FF9933) để làm nổi bật thông tin và hành động quan trọng.
- Phông chữ: Phông chữ sans-serif 'Inter' cho cả tiêu đề và nội dung văn bản do vẻ ngoài hiện đại và trung tính, đảm bảo dễ đọc trong môi trường chuyên nghiệp.
- Sử dụng các biểu tượng đơn giản, rõ ràng để biểu thị các hành động như tải lên, phân tích và xem báo cáo.
- Thiết kế bố cục rõ ràng và trực quan để dễ dàng điều hướng, có các nút nổi bật cho các hành động chính.
- Kết hợp các hiệu ứng hoạt hình tinh tế trong quá trình phân tích để cung cấp phản hồi cho người dùng và cho biết tiến trình.
